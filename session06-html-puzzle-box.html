  • Preparação

    Imprima pelo menos 12 cópias do modelo de quebra-cabeça em cubos.

    Criar um quebra-cabeça em cubo é muito simples. Mas, para ter cubos os suficiente para jogar, você vai precisar de pelo menos 12 cubos – Para garantir que você tenha cubos suficientes para abrir e fechar os marcadores. No total, para um par de jogadores, você precisa fazer 24 cubos. Então, para construir 24 cubos, você precisa imprimir 12 modelos de papel para montar.

    É recomendado que imprima o quebra-cabeça em papel grosso (cartolina, papelão etc), assim vai ser mais fácil montar e vai ajudar na construção

    Se você tem um tempo limitado, você pode fazer essa atividade cortando pedados de papel ou usando post-its no lugar das caixas.

  • Introdução: Fazendo os cubos

    15 minutos

    Isso é muito divertido, mas demora um pouco. Primeiros os alunos montam suas caixas. Bônus: Se o tempo permitir, o mentor pode jogar HTML Corners usando os cubos parar sinalizar as estações.





  • Desafio do quebra-cabeças HTML em cubos

    20 minutos

    Primeiro monte marcadores na ordem certa em um pequeno grupo. Você pode modelar esta montagem para todo o grupo. Em seguida, faça com que os alunos ou grupos peguem suas caixas e "subam" na frente deles. Diga aos participantes que seu novo objetivo é empilhar os blocos na ordem correta para a construção de uma página da web. Então deixe-os competir para ver quem pode construir uma página da web o mais rápido!



    Essa é a combinação certa da estrutura dos marcadores HTML.

    <html>
      <head>
        <title>
        </title>
        <style>
        </style>
      </head>
      <body>
        <p>
        </p>
      </body>
    </html>

  • Reflexão: Revisão dos marcadores

    10 minutos

    Repita o HTML Corners se o tempo permitir. Você também pode pedir para os alunos gritarem o que cada marcador significa.

    Bônus: Se o tempo permitir, mostre os marcadores clicando em "Ver código" ou usando o X-Ray Goggles em uma página web.
